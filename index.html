What is this?

This is an interpreter for a 1970's era programming language called Chip8 that was used on hobby computers for games like Pong, Space Invaders, etc. My program reads the same programs as the old computers did and emulates how they operated, letting you experience the programs in your browser.

I got the idea from a blog article and had a blast putting it together. Check out my code on github and let let me know if you have any issues or suggestions.
